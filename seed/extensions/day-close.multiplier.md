# Day Close Extension: Мультипликатор IWE

> **Точка подключения:** шаг 8 Day Close.
> **Управление:** удалить этот файл, если мультипликатор не нужен.

## Алгоритм

- **WakaTime** — физическое время за день (`fetch-wakatime.sh` запрашивает через API).
- **Бюджет закрыт** — сумма бюджетов по ВСЕМ РП за день.
  Бюджет каждого РП — из `WP-REGISTRY.md` (колонка «Бюджет, h»). Если бюджет не указан → 0.25h.
  - done → полный бюджет (или пропорционально фазам для зонтичных)
  - partial → % выполнения × бюджет
  - not started → 0h
- **Мультипликатор дня** = Бюджет закрыт / WakaTime. Формат: `N.Nx`.

## Действия

1. Открыть `DS-strategy/docs/WP-REGISTRY.md` — найти все РП со статусом `done` или `partial` за сегодня.
   Суммировать их бюджеты (колонка «Бюджет, h»). РП без бюджета → по 0.25h каждый.
   Получить `<бюджет>` в часах (десятичное число, напр. `3.5`).

2. Вызвать скрипт напрямую (slash-команды недоступны внутри скилла):
   ```bash
   bash "${CLAUDE_SKILL_DIR}/../wakatime/scripts/fetch-wakatime.sh" multiplier --budget <бюджет>
   ```
   Скрипт выведет готовую markdown-таблицу мультипликатора.

Таблица мультипликатора находится в контексте сессии.
